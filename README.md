# 🚀 StaffPe - Staff Management SaaS Website & Admin Panel

A modern, responsive SaaS-based staff management platform built using React Native Web & Expo. This project includes a professional landing page along with an interactive admin dashboard for managing staff operations efficiently.

---

## 🌐 Live Demo

👉 https://staff-management-site.vercel.app/

---

## 📌 Features

### 🔹 Landing Page

* Modern SaaS UI design
* Fully responsive (Mobile + Tablet + Desktop)
* Hero section with CTA
* Features showcase
* Pricing plans
* FAQ section
* Contact form

### 🔹 Admin Dashboard

* Secure Login System
* Drawer Navigation UI
* Project / Task Management
* Add & manage team members
* Search & filter functionality
* Responsive dashboard layout

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/staff-management-ui.git
cd staff-management-ui
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Run locally

```
npx expo start
```

---

## 🚀 Deployment (Vercel)

```
npx expo export
```

Then deploy the `dist` folder on Vercel.
